# 写个python

读取程序所在文件夹下的excell文件
没有标题行
从A、B列第二行开始
在指定文件夹A下（包括子文件夹下）搜索文件夹名与A列从第二行开始的单元格内的内容相同的文件夹
移动在指定文件夹B下（包含子文件夹下）搜索文件夹名与B列从第二行开始的单元格内的内容相同的文件夹内

同时支持xls和xlsx格式

指定文件夹A和B路径由用户自行输入
